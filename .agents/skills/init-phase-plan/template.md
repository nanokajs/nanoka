## 着手前チェック（再開時用）

新規セッションで本計画から再開する場合、以下を確認:

1. `git status` と `git log` で前回の到達点を把握
2. 本ファイルのチェックボックスと実コードの差分を照合（チェックが正しいか）
3. `docs/backlog.md` に新規項目が増えていないか確認（増えていれば本計画への取り込み要否を判断）
4. 直近完了マイルストーンの「完了基準」を実行して回帰を検出
